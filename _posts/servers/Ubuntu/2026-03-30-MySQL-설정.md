# 우분투에서 MySQL 설정

## 기본 패키지 설치

```shell
apt install mysql-server mysql-client
```

## 디렉토리 이동
```shell
vi /etc/apparmor.d/tunables/alias
alias /var/lib/mysql -> /datbase/mysql/data
chown -R mysql:mysql /database/mysql
```

## 다른 곳에서 접속 허용
```shell
vi /etc/mysql/mysql.conf.d/mysqld.cnf
bind-address            = 0.0.0.0
```

## root 비번 변경
> ALTER USER \`root\`@\`localhost\` IDENTIFIED WITH 'caching_sha2_password' BY \`비밀번호\`;  
> systemctl restart mysql


## 사용자 계정 생성
```sql
-- 테이블 생성
CREATE DATABASE `{데이타베이스명}`

-- 사용자 생성
CREATE USER `{사용자ID}`@`{접속허용호스트}` IDENTIFIED BY `{비밀번호}`;

-- 권한 설정
GRANT ALL PRIVILEGES ON `{데이타베이스명}`.* TO `{사용자ID}`@`{접속허용호스트}` WITH GRANT OPTION;
```
