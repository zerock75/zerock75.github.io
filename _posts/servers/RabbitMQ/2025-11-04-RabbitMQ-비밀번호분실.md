# Windows + RabbitMQ 비밀번호 분실

## 1. 관리자권한으로 CMD 실행

## 2. RabbitMQ 설치 경로로 이동
```cmd
cd "C:\Program Files\RabbitMQ Server\rabbitmq_server-{버전}\sbin"
```

## 3. RabbitMQ 중지
```cmd
net stop RabbitMQ
```

## 4. 현재 사용자 쿠키 내용 확인
```cmd
type "C:\Users\{계정}\.erlang.cookie"
PJLYDPFBKTRMPDLSQKVI    # 내용 복사
```

## 5. 기존 쿠키 읽기전용 해제
"C:\Windows\System32\config\systemprofile\.erlang.cookie" 위치에 있는 파일의 속성에서 **읽기전용** 해제

## 6. 새로운 쿠기 복사
4에서 확인한 쿠키를 5의 파일로 교체한다.

## 7. 5의 .erlang.cookie 파일의 **읽기전용** 다시 설정

## 8. RabbitMQ 시작
```cmd
net start RabbitMQ
```

## 9. 계정 추가
```cmd
rabbitmqctl.bat add_user {계정아이디} {비밀번호}
```

## 10. 관리자 태그 및 권한 부여
```cmd
rabbitmqctl.bat set_user_tags {계정아이디} administrator
rabbitmqctl.bat set_permissions -p / {계정아이디} ".*" ".*" ".*"
```
