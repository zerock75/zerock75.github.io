### 네임드 볼륨에 리포지토리 생성
```cmd
 docker run -d \ 
  --name docker-registry \ 
  --restart=always  -p 5000:5000 \ 
  -v registry-data:/var/lib/registry \ 
  registry:2
```

### HTTP push 오류 해결 
각 클라이언트의 Docker 설정에 아래를 추가.
Docker -> Settings -> Docker Engine
```
{
  "insecure-registries": ["레지스트리서버IP:5000"]
}
```

```bash
# 레지스트리 주소로 태그
docker tag hello-world 192.168.0.232:5000/hello-world:test
# 푸시
docker push 192.168.0.232:5000/hello-world:test
# 카탈로그에 나타나는지 확인
curl http://192.168.0.232:5000/v2/_catalog
```
