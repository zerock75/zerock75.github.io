## 1. 현재 설치된 WSL 확인
먼저 내 PC에 어떤 리눅스가 깔려 있는지 확인해야 한다.
 `PowerShell`을 관리자 권한으로 실행한다.
```powershell
wsl -l -v
-------------------------------------------------
  NAME              STATE           VERSION
* docker-desktop    Stopped         2

```
위와 같이 **Ubuntu**가 목록에 없다면 먼저 **Ubuntu**를 설치해 줘야 한다.


## 2. Ubuntu 설치
```powershell
wsl --install -d Ubuntu
```

## 3. WSL 진입
이제 다운받은 우분투 안으로 진입한다.
```powershell
wsl -d Ubuntu
```

## 4. Systemd 활성화
```shell
sudo nano /etc/wsl.conf
------------------------------
[boot]
systemd=true
```

### WSL 재부팅
`Ubuntu`를 빠져나온 후 리눅스를 강제로 껐다 켠다.
```powershell
wsl --shutdown
```



## 5. NodeJS 및 OpenClaw 설치
 1. 패키지 목록 업데이트
 ```shell
 sudo apt-get update && sudo apt-get install -y curl
 ```

 2. Node.js 24 설정
 ```shell
 curl -fsSL https://deb.nodesource.com/setup_24.x | sudo -E bash -
 ```

 3. Node.js 설치
 ```shell
 sudo apt-get install -y nodejs
 ```

 4. OpenClaw 설치
 ```shell
 # OpenClaw 전역 설치
 sudo npm install -g openclaw@latest

 # 설치 확인
 openclaw --version
 ```

 5. 초기 설정
 ```shell
 openclaw onboard
 ```
