## 4. 블로그에 Jekyll을 설치.
VSCode에서 블로그를 연 후 터미널을 실행한다.
```terminal
bundle init
bundle config set path 'vendor/bundle'
bundle add jekyll
bundle exec jekyll new --force --skip-bundle .
```

## 5. 서버작동
```terminal
bundle exec jekyll serve --livereload
```
> &minus;&minus;livereload 옵션을 추가하면 자동으로 새로고침 된다.
{: .prompt-tip }


## 6. index 파일 교체
