## 아두이노를 활용한 멀티탭 제어 기본 회로도
![기본회로도](/imgs/develop/etcs/arduino-muititab-on_off-diagram.png)

## 아두이노에 프로그램 업로드
1. Arduino IDE 설치: [아두이노 공식 홈페이지](https://www.arduino.cc/en/software/){:target="_blank"}에서 다운받아 설치한다.
2. PC 연결: USB 케이블로 아두이노와 PC를 연결한다.
3. 설정: IDE 메뉴에서 다음을 확인한다.
 - 툴 > 보드: Arduino Uno(사용 중인 보드 모델 선택)
 - 툴 > 포트: 아두이노가 연결된 COM 포트 선택
4. 코드 작성 및 업로드: 아래 제어 코드(C++)를 복사해 붙여넣은 후, 왼쪽 상단의 **화살표 모양 아이콘(업로드)**을 클릭한다.

## 제어 코드(C++)
```c++
int relayPin = 7; // 릴레이 신호선(SIG)이 연결된 핀 번호

void setup() {
  pinMode(relayPin, OUTPUT); // 7번 핀을 출력 모드로 설정
  digitalWrite(relayPin, LOW); // 초기 상태는 OFF (릴레이 모듈 특성에 따라 HIGH일 수도 있음)
  Serial.begin(9600);       // PC와의 시리얼 통신 시작
  Serial.println("Relay Control Ready. (1: ON / 0: OFF)");
}

void loop() {
  if (Serial.available() > 0) {
    char command = Serial.read(); // PC로부터 명령 읽기

    if (command == '1') {
      digitalWrite(relayPin, HIGH); // 릴레이 작동 (멀티탭 ON)
      Serial.println("Power ON");
    } 
    else if (command == '0') {
      digitalWrite(relayPin, LOW);  // 릴레이 해제 (멀티탭 OFF)
      Serial.println("Power OFF");
    }
    else if (command == '2') { // '2'가 들어오면 현재 상태 확인
      // 디지털 핀의 현재 출력 상태를 읽어옴
      int currentState = digitalRead(relayPin);
      if (currentState == HIGH) {
        Serial.println("ON");
      } else {
        Serial.println("OFF");
      }
    }
  }
}
```

## 회로 및 동작 참고사항
### **Active Low vs High**: 사용하는 릴레이 모듈에 따라 `HIGH` 일 때 켜지는 제품이 있고, `LOW`일 때 켜지는 제품이 있다.  
 이를 구별하는 방법은 아두이노에서 **5V(HIGH)** 신호를 보낼 때 릴레이가 '딸깍'하며 켜지면 `HIGH` 제품이고, 반대로 **0V(LOW)** 신호를 보낼 때 릴레이가 '딸깍'하며 켜지면 `LOW` 제품이다.
 만약 반대로 동작한다면 **제어 코드**의 `HIGH`와 `LOW`를 서로 바꾼다.
 ```c++
// Active Low 방식 릴레이용 코드 예시
if (command == '1') {
  digitalWrite(relayPin, LOW);  // 전압을 낮춰야(LOW) 릴레이가 켜짐(ON)
  Serial.println("Power ON");
} 
else if (command == '0') {
  digitalWrite(relayPin, HIGH); // 전압을 높여야(HIGH) 릴레이가 꺼짐(OFF)
  Serial.println("Power OFF");
}
 ```
내 릴레이 방식 확인 방법
 1. 외관확인: 모듈에 **Low Level Trigger**라고 적혀 있으면 **Active Low** 방식이다.
 2. 동작확인: `제어 코드`를 실행했는데 '1'을 입력했을 때 꺼지고, '0'을 입력했을 때 켜진다면 **Active Low** 방식의 모듈이다.

### 시리얼 모니터 확인
업로드 완료 후 IDE 오른쪽 상단의 돋보기 아이콘(**시리얼 모니터**)을 누르고, 전송창에 **1** 또는 **0**을 입력해 보라.  
릴레이에서 '딸깍' 소리가 나면 성공이다.
