## 가상환경 생성 및 활성화
```cmd
# 가상환경 생성
python -m venv .venv

# 활성화(Linux)
source .venv/bin/activate

# 활성화(Windows)
.venv\Scripts\activate
```

## 설치된 패키지 목록 저장
```
pip freeze > requirements.txt
```

## 가상환경 비활성화
```
deactivate
```
