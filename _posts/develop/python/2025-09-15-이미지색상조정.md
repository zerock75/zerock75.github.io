1. 이미지 밝기 조절

1은 발기 100%를 의미함.
여기에 +- % 를 더해서 120, -120% 로 밝기를 조절할 수 있다.
```py
  from PIL import Image, ImageEnhance

  _BRIGHTNESS = 20

  image = Image.open("sample.jpg")
  enhancer = ImageEnhance.Brightness(image)
  enhancer_value = 1 + ( _BRIGHTNESS / 100)
  brightened_image = enhancer.enhance(enhancer_value)
```

3. 선명도 조절
```py
  from PIL import Image, ImageEnhance

  _SHARPNESS = 10

  image = Image.open("sample.jpg")
  enhancer_value = max(0.0, 1.0 + (_SHARPNESS / 100.0))

  enhancer = ImageEnhance.Sharpness(image)
  sharpened_image = enhancer.enhance(enhancer_value)
```

5. 따뜻함
```py
  from PIL import Image
  import numpy as np

  _WARMH  = 20

  image = Image.open("sample.jpg")
  image = image.convert("RGBA") 
  r, g, b, a = image.split()

  r_array = np.array(r, dtype=np.float32)
  b_array = np.array(b, dtype=np.float32)

  adjustment = _WARMH / 100 * 100
  r_array = np.clip(r_array + adjustment, 0, 255) 
  b_array = np.clip(b_array - adjustment, 0, 255) 

  r = Image.fromarray(r_array.astype("uint8"))
  b = Image.fromarray(b_array.astype("uint8"))
  warmed_image = Image.merge("RGBA", (r, g, b, a))
```

6. 색조 변환
```py
  from PIL import Image
  import numpy as np
  import cv2

  _HUE = 20

  np_image = np.array(image.convert("RGBA"))  # RGBA 유지
  rgb_image = np_image[..., :3]  # RGB 채널만 추출
  alpha_channel = np_image[..., 3]  # 알파 채널 추출

  # RGB → HSV 변환
  hsv_image = cv2.cvtColor(rgb_image, cv2.COLOR_RGB2HSV).astype(np.float32)

  # Hue 값 조정 (OpenCV Hue 범위는 0~180)
  hue_adjust = _HUE * 0.5  # -100 ~ 100 범위를 -90 ~ 90으로 조정
  hsv_image[..., 0] = (hsv_image[..., 0] + hue_adjust) % 180  # Hue 값 조정 및 순환 처리

  # HSV → RGB 변환
  adjusted_rgb = cv2.cvtColor(hsv_image.astype(np.uint8), cv2.COLOR_HSV2RGB)

  # RGB와 알파 채널 병합
  adjusted_image = np.dstack((adjusted_rgb, alpha_channel))

  # NumPy 배열 → Pillow 이미지로 변환
  final_image = Image.fromarray(adjusted_image.astype(np.uint8), "RGBA")
```
