#### Windows Form vs WPF: UI 기술 차이
##### Windows Forms:
  - 특징: Windows의 전통적인 UI(버튼, 입력상자 등)를 그대로 가져오 ㅏ사용하는 '컨트롤' 기반 방식
  - 장점: 배우기 쉬움.
  - 단점: UI 디자인을 자유롭게 꾸미기 어려움. 애니메이션, 투명도, 3D등을 적용하기 어려움

##### WPF:
  - 특징: UI를 XAML이라는 XML 기반언어로 설계. 데이터와 UI를 분리하는데 특화. 그래픽을 DirectX를 통해 직접 그림
  - 장점: 디자인 자유도가 높음
  - 단점: 배우기 어려움.
