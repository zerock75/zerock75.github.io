## 0. 기본정보
1. docker 실행, vscode
2. laravel10, php8.2, mysql8.0


## 1. WSL 진입
```console
wsl -d <Distro> -u <사용자명>
```

## 2. 위치이동
작업할 디렉토리로 이동한다. (ex. /home )
```shell
cd /home
```

## 3. 프로젝트를 생성한다
```shell
curl -s "https://laravel.build/<프로젝트명>?with=mysql,redis" | bash
```
